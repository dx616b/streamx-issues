# StreamX Issues

Public issue tracker for **StreamX**.

Use this repository to:

- Report bugs
- Request features
- Ask usage/support questions

## Before Opening an Issue

- Search existing issues first to avoid duplicates.
- Use the appropriate issue template.
- Include enough detail so the issue can be reproduced.

## What to Include

For bug reports, please include:

- StreamX version
- Relevant config (remove secrets/tokens)
- Steps to reproduce
- Expected behavior vs actual behavior
- Logs/screenshots (if available)

## Security

Please **do not** post secrets, API tokens, or private credentials.

If you need to report a security issue, contact maintainers privately instead of opening a public issue.

## Source Code

The StreamX source code is maintained in a private repository.  
This repository is for public tracking and communication only.
